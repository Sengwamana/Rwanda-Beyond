# Smart Maize Farming System - End User Guide

## What this project does
Smart Maize is a farming assistant for maize growers. It combines sensor readings, weather data, and AI analysis to help with:
- Irrigation timing
- Pest detection (especially fall armyworm)
- Fertilizer and farm-care recommendations
- Alerts through web and mobile channels

## Who should use it
- Farmers managing one or more maize farms
- Field experts supporting farmers
- Administrators managing users, devices, and system health

## Main features for farmers
- Farm dashboard with farm status and key metrics
- Sensor monitoring (soil moisture, temperature, humidity, and related data)
- Weather overview and forecast per farm
- Recommendation feed with priority and status
- Pest image upload and AI-assisted analysis
- Multilingual support (English, Kinyarwanda, French in several areas)

## Typical user flow
1. Sign up or log in.
2. Open your dashboard.
3. Select a farm.
4. Review sensor and weather conditions.
5. Read recommendations and respond (accept, reject, defer).
6. If pests are suspected, upload an image for analysis.
7. Follow alerts and keep checking updates.

## Alerts and notifications
The system can notify you when:
- Soil moisture is too low or too high
- Weather risk may affect irrigation or treatment
- A new recommendation is generated
- A pest detection requires action

## USSD and SMS access
For users with feature phones, the platform includes USSD and SMS flows for:
- Checking recommendations
- Viewing basic farm status
- Getting weather snapshots
- Navigating key actions without a smartphone

## User roles
- Farmer: accesses own farms and recommendations
- Expert: reviews and supports broader farm activity
- Admin: manages platform settings, users, devices, and reports

## Safety and data notes
- Keep your login credentials private.
- Only upload images that belong to your farm context.
- Contact admin support if alerts seem incorrect or delayed.

## If something is not working
- Re-login if your session expires.
- Check internet connection.
- Confirm your farm has active sensors.
- Contact support/admin if recommendations or sensor updates stop.
