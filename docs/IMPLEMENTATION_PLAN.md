# System Functionalities

## 1. Real-Time Environmental Data Collection

The system must collect environmental data from IoT sensors deployed on the farm.

Functions:

- Collect soil moisture data.
- Collect soil temperature data.
- Collect air humidity data.
- Collect soil nutrient data (NPK levels).
- Capture crop images for pest detection.
- Record timestamps of sensor readings.

## 2. Sensor Data Transmission

The system must transmit collected data from the field to the processing platform.

Functions:

- Send sensor readings via GSM communication module.
- Transmit data from microcontroller to the cloud server.
- Ensure continuous data synchronization between farm devices and server.

## 3. Data Storage and Management

The system must store and manage collected data.

Functions:

- Store sensor data in the database.
- Store pest detection images.
- Store farm information.
- Store user profiles (farmers, experts, admin).
- Maintain historical environmental data.
- Maintain system logs.

## 4. AI-Driven Data Analysis

The system must analyze collected data using artificial intelligence.

Functions:

- Analyze soil moisture trends.
- Predict irrigation requirements.
- Analyze environmental conditions.
- Detect Fall Armyworm (FAW) using image recognition.
- Evaluate pest risk levels.
- Analyze nutrient status for fertilization guidance.

## 5. Smart Irrigation Decision Support

The system must determine irrigation needs automatically.

Functions:

- Evaluate soil moisture thresholds.
- Determine irrigation scheduling.
- Recommend irrigation amount.
- Generate irrigation alerts to farmers.
- Trigger automated irrigation if conditions require it.

## 6. Pest Detection and Management

The system must support early pest detection and management.

Functions:

- Detect Fall Armyworm from uploaded images.
- Predict pest outbreak risk using weather data.
- Generate pest alerts.
- Recommend pest control actions.
- Schedule pest control operations.

## 7. Fertilization Management

The system must provide intelligent fertilization recommendations.

Functions:

- Analyze soil nutrient levels (NPK).
- Generate fertilization schedules.
- Recommend fertilizer type and quantity.
- Send fertilization alerts to farmers.

## 8. Farmer Notification System

The system must communicate with farmers.

Functions:

- Send irrigation alerts via SMS.
- Send pest detection alerts via SMS.
- Send fertilization recommendations via SMS.
- Send system notifications through USSD services.
- Deliver real-time advisory messages.

## 9. Farmer Interaction and Feedback

Farmers must be able to interact with the system.

Functions:

- Receive alerts and recommendations.
- Approve irrigation actions.
- Postpone irrigation schedules.
- Submit crop images for pest detection.
- Report farm issues.

## 10. Expert Support and Monitoring

Agricultural experts must be able to interact with the system.

Functions:

- Review sensor data.
- Analyze pest detection results.
- Provide expert recommendations.
- Update pest control strategies.

## 11. System Administration

The system administrator manages overall system operation.

Functions:

- Manage user accounts.
- Configure system settings.
- Monitor system performance.
- Manage farm data.
- Monitor IoT devices.

## 12. Action Execution and Logging

The system must execute actions and maintain records.

Functions:

- Execute irrigation schedules.
- Execute pest control actions.
- Execute fertilization schedules.
- Log all system actions.
- Maintain history of farm operations.

## 13. Continuous Monitoring

The system must support continuous monitoring.

Functions:

- Monitor farm conditions in real time.
- Re-evaluate conditions periodically.
- Generate new alerts when thresholds are exceeded.
- Maintain ongoing monitoring loop.

## 14. Dashboard and Reporting

The system must present farm data through an interface.

Functions:

- Display farm sensor data on dashboard.
- Display irrigation schedules.
- Display pest detection results.
- Display system activity logs.
- Generate farm reports.
